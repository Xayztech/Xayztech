Proteksi Anti-Rusuh Panel Pterodactyl

Amankan panel Anda dari tindakan yang tidak diinginkan dengan satu baris perintah.

Apa Fungsinya?

Skrip ini adalah sebuah alat keamanan yang dirancang khusus untuk Pterodactyl Panel. Tujuannya adalah untuk membatasi aksi-aksi destruktif yang bisa dilakukan oleh admin selain Admin Utama (User ID 1). Ini sangat berguna jika Anda memiliki beberapa admin dan ingin mencegah kesalahan atau penyalahgunaan wewenang.

Cara Pemasangan

Cukup masuk ke terminal server Anda sebagai root, lalu salin dan jalankan perintah di bawah ini. 

bash <(curl -s https://xayztech-installasi-fitur-anti-rusuh.vercel.app/Protection-Pterodactyl.sh)

Instruksi Skrip

Setelah dieksekusi, Anda akan disambut oleh menu interaktif:

  1. Pasang Fitur Anti Rusuh: Memulai instalasi. Skrip akan otomatis mengecek & membuat backup sebelum memodifikasi panel.
  2. Lepas Fitur Anti Rusuh (Restore Panel): Mengembalikan panel ke kondisi semula dengan memilih file backup yang tersedia.
  3. Buat Backup Manual: Hanya menjalankan fungsi backup tanpa instalasi.
  4. Keluar: Menutup skrip.

Peringatan Penting

Jika Anda melakukan update Pterodactyl Panel, file-file yang telah dimodifikasi akan kembali ke semula. Ini berarti fitur proteksi akan hilang. Anda perlu menjalankan ulang perintah instalasi ini setelah update.
